STATIC elements are STUCK
in the normal pageflow.
They have NO coordinates
and can NOT be positioned!
RELATIVE elements can be
OFFSET from their PARENT
by specifiying coordinates
but still remain within the normal page flow!
ABSOLUTE elements are REMOVED
from the normal flow and can be
placed ANY WHERE with coordinates
offset from the document itself!
FIXED elements are like absolute elements that are relative to the VIEWPORT
instead of the document, so they don't scroll as the page scrolls!
